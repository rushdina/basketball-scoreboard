# 🏀 Basketball Scoreboard

A basketball scoreboard that allows users to track scores for the Home and Guest teams. Each button updates the score correctly, and a New Game button resets the scoreboard.

## 🖼️ Preview
🔗 Live Demo: https://basketball-scoreboard-rushdina.netlify.app/ 

![Basketball Scoreboard Preview](images/screenshot-basketballScoreboard.png)

## 🛠️ Technologies Used
- **Frontend:** `HTML`, `CSS`, `JavaScript`

## 📚 What I Learned
- How to store and manage state using JS objects
- Updating DOM dynamically based on user interactions
- Using data attributes (`data-team`, `data-points`) for reusable buttons
- Event delegation for handling multiple buttons
- Resetting game state without duplicating logic

## 💡 Future Improvements
- Highlight leading team
- Add game period, fouls, or timer counter
