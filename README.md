# 🏀 Basketball Scoreboard

A basketball-style scoreboard solo project from the [Scrimba Frontend Developer Career Path](https://scrimba.com/frontend-path-c0j).  
Tracks Home and Guest scores in real time, allowing increments by 1, 2, or 3 points and a full reset.

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- JavaScript (ES6)

## 🚀 Features
- Real-time score updates in the DOM.
- Reusable function for all score increments.
- Reset game to zero with one button.
- Efficient event delegation for button clicks.
- Cached DOM elements for faster updates.

## 🧩 Concepts Practiced
- DOM manipulation for dynamic updates.
- Event delegation and dataset usage.
- State management with JS objects.
- Reusable functions for cleaner code.
- Flexbox layout and custom font integration.

## 💡 Future Improvements
- Highlight leading team dynamically.
- Save scores in localStorage.
- Track game period, fouls, and timer.
- Dark/light mode toggle.
